# 第一步：建立一个rails应用
## 命令解析
`mkdir`是建立一个文件夹，后面跟一个字符串当文件夹名

`cd`是进入某一个文件夹中，可以用相对路径，也可以用绝对路径

`pwd`可以查看当前目录的绝对路径

`rails new`命令可以帮助你快速建立一个rails项目

`rails server`用来启动当前目录下的rails项目

## 这一步所需操作
```
mkdir projects
cd projects
rails new railsgirls
cd railsgirls
rails server
```
完成这些步骤后，注意看命令行的提示信息，访问打印出来的URL就可以打开你的第一个rails应用啦
